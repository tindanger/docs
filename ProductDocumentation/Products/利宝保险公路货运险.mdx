## 1. 产品介绍
公路货物运输保险（公路货运险）采用"仓至仓"责任条款，保险责任自货物离开起运地最后一个仓库起，至目的地收货人第一个仓库终止。覆盖自然灾害、意外事故导致的直接损失及合理施救费用，有效降低物流企业运输风险，增强货主信任。

## 2. 保障责任

### 2.1 主险责任
#### 基本险：
- 自然灾害：火灾、爆炸、雷电、暴雨等
- 运输工具事故：碰撞、倾覆、沉没等
- 装卸操作损失（非包装/操作问题）
- 共同海损费用
- 施救保护费用

#### 综合险（含基本险+）：
- 震动/挤压导致的货物破碎、渗漏
- 盗窃/整件提货不着
- 合规雨淋损失

### 2.2 附加险责任
| 附加险种 | 保障内容 | 限制条件 |
|---------|---------|---------|
| 盗抢险 | 运输途中抢劫/偷窃损失 | 单次限额≤50%且≤80万 |
| 特殊车辆责任 | 临牌/超牌重型平板车 | 需合规装载 |
| 冷藏品扩展 | 非疫苗冷藏品运输 | 不保设备故障损失 |

## 3. 投保规则
### 承保范围：
- **地域**：中国大陆（不含黑吉辽鲁津苏部分城市及港澳台），西藏/新疆/青海可特约扩展
- **包装**：符合国标/行标专业包装（不保散装/液体/粉末）
- **运输工具**：集装箱/全封闭厢式卡车，或雨布全覆卡车

### 不承保货物：
```
1. 贵金属/艺术品/奢侈品
2. 动植物/易腐品/鲜活货
3. 工程机械/汽车配件/精密仪器（＞200万）
4. 危险品/药品原料/疫苗
5. 散装货物/私人货物/展品
```

## 4. 适用条款
- 《国内水路、陆路货物运输保险条款》
- 《承运人、托管人或第三方责任》
- 《神秘失踪除外》《30天取消条款》
- 《附加传染病免责》《窃盗保险扩展条款-A》

## 5. 投保须知
### 投保资料：
1. 营业执照+财务报表（三选一）
2. 历史保单（近3年）
3. 开票资料
4. 风勘问卷（含运输路线/车辆配置/客户信息）

### 保费计算：
```
最终保费 = max(基础费率×调整系数, 2.3万) ×核保系数
```

| 调整类型       | 系数范围     | 示例条件                |

| 免赔额调整     | 0.9-1.12    | 免赔2万→0.9           |

## 9. 特别约定
1. **特殊货物**：
   - 电子设备：单价≤5000元，不保盗窃
   - 摩托车：单价≤1万，需集装箱运输
   - 冷藏品：需提供温度/车辆信息

2. **区域扩展**：
   - 西藏/新疆/青海需单独约定

3. **赔偿限制**：
   - 酒类：单瓶最高赔500元
   - 汽车配件：需第三方鉴定残值
