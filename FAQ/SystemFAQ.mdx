## 1. 投保相关

### 1.1 如何选择险种？
1. 进入【产品中心】查看按公司/分类筛选的保险产品
2. 使用关键字搜索特定产品（如"雇主责任险"）
3. 查看产品详情页的保障范围、费率说明、投保要求
4. 点击"生成报价"进行方案试算

### 1.2 如何选择保障方案？
1. 根据企业规模选择保额梯度（50万/80万/100万）
2. 按行业风险等级匹配对应方案（建筑行业选高风险方案）
3. 参考系统推荐的热门方案配置
4. 可联系客服获取定制化方案建议

### 1.3 如何填写投保页面？
1. 在订单创建页面逐项填写、勾选或下拉：投保人信息、被保险人信息、保险期限、保障方案等基本要素
2. 人员类信息支持Excel模板导入人员清单，先下载模板，按模板要求填写人员信息，再上传模板文件，系统会自动导入人员信息
3. 营业执照或其他附件材料需要在投保申请时上传
4. 系统会实时校验填写信息，确保准确性
5. 填写完成后点击【提交】按钮，提交后跳转至该订单工作台页面，审核通过后可点击【下载投保单】区域进行下载格式化投保单

### 1.4 如何下载和上传投保单？
1. 订单提交后会跳转到该订单工作台页面，审核通过可在【订单操作】模块点击【下载投保单】区域进行下载格式化投保单
2. 在【订单管理】模块中找到订单，点击【转工作台】,在工作台模块中点击【订单详情】，在【标的信息】——【保险人】下方找到格式投保单，或【订单资料】栏找到格式投保单
3. 下载投保单打印后需要投保人盖章
4. 盖章后扫描上传至【订单管理】——【转工作台】——在页面下方【订单操作】区域——点击【上传投保单盖章影像件】模块进行上传，或直接将投保单文件拖动到【上传投保单盖章影像件】模块
5. 在订单详情页【欠缴资料】模块支持补传投保单盖章件

### 1.5 投保申请时需要填写哪些信息？
- 企业基本信息：公司名称、统一社会信用代码、注册地址
- 保险要素：保障方案、保险起期、参保人数
- 账户信息：对公账户（用于退保/批改退款）
- 联系人信息：姓名、手机、邮箱

### 1.6 投保申请时需要上传哪些材料？
- 基础材料：加盖公章的投保单
- 人员清单：参保人员姓名、身份证号、工种等，以保险公司要求为准
- 特殊材料：特殊高危行业可能需安全生产许可证，建筑行业可能需建筑施工许可证和建筑工程合同等材料，其他行业可能需提供相关资质证明，以保险公司要求为准

## 2. 批改相关

### 2.1 如何进行保单批改？
1. 在订单详情页点击【转工作台】，在【订单操作】模块点击【申请批改】（需要订单已完成支付并通过投保单影像件审核）
2. 下载并按模板要求填写批改申请模板
3. 上传批改模板
4. 等待核保通过后完成费用结算
5. 退保/减人/信息变更/延期等特殊批改须联系聚千保平台客服开通权限

### 2.2 批改申请需要哪些材料？
普通批改（增人/替换人员）：
- 批改申请书盖章影像件（如订单已指定授权人批改，则无需上传）
- 变更证明材料（如人员离职证明）
非标批改（退保/减人/信息变更/延期）：
- 批改申请书盖章影像件及物理件
- 变更证明材料（如人员离职证明）
- 最新营业执照（涉及主体变更时）
- 建工延期情况声明书（延期批改时上传）

### 2.3 批改处理时效是多久？
- 普通批改（在线API产品）：次日零时生效
- 普通批改（非在线API产品）或非标批改：3-5个工作日
- 退保批改：需财务复核（5-7个工作日）
- 可实时查看【未完成批改】追踪进度

## 3. 保单相关

### 3.1 如何下载电子保单？
1. 在【订单管理】找到对应订单，转到工作台，在下方【订单操作】模块，直接点击【查看保单】按钮（承保完成后出现）
2. 点击【订单详情】页在【标的信息】栏——【保险人】下方点击【关联保单号】
3. 点击【订单详情】页在【订单资料】栏找到【电子保单】

### 3.2 如何申请纸质保单？
1. 在订单详情页点击"申请纸质保单"
2. 填写邮寄地址和联系人
3. 支付快递费用（到付或在线支付）
4. 在【物理件追踪】查看寄送进度

## 4. 团队管理

### 4.1 如何创建子账号？
1. 进入【团队管理】-【创建子账号】
2. 填写姓名、手机、邮箱等基本信息
3. 设置账号权限（产品可见范围/操作权限）
4. 系统自动发送激活邮件给子账号

## 5. 结算管理

### 5.1 如何申请积分结算？
1. 进入【结算管理】-【积分申请】
2. 选择结算方式（现金/普通发票结算/增值税发票结算）
3. 上传盖章的结算申请单
4. 填写收款账户信息并提交审核

### 5.2 积分结算规则是什么？
- 每月16日-25日可申请兑换上月积分，我们将会在每月20日起开始推广费结算。
- 积分申请 选择现金兑换仅支持收款人为个人，收款人为公司请在积分兑换申请单审核通过后7个工作日内开具发票，我司收到发票后5个工作日内进行打款
- 选择不提供发票或者增值税普通发票，兑换积分比例不变，选择增值税专用发票1%，兑换积分比例追加0.5%；选择增值税专用发票3%，兑换积分比例追加1%；选择增值税专用发票6%，兑换积分比例追加2%。
- 请仔细核对收款人户名、开户行、银行账户，并且仔细核对积分明细，积分兑换申请单一经审核通过，将不再支持修改。

## 6. 营销分析

### 6.1 如何查看销售数据？
1. 首页查看实时业绩看板
2. 【营销分析】查看趋势图表
3. 下载不同维度的数据报表
4. 设置自定义数据监控看板

## 7. 个人设置

### 7.1 安全设置
1. 密码修改：至少包含大小写字母+数字
2. 绑定微信：支持扫码登录和消息提醒
3. 密保手机：用于重要操作验证
4. 登录记录：查看最近3个月登录历史

### 7.2 个性化设置
1. 界面主题：深色/浅色模式切换
2. 消息通知：设置邮件/短信提醒偏好
3. 快捷入口：自定义首页功能模块排序
4. 数据看板：配置常用统计指标

### 7.3 操作员设置
1. 在【个人设置】-【操作员设置】-【新增操作员】
2. 根据页面提示填写操作员信息
3. 设置操作员权限，包括产品可见范围、操作权限等
4. 系统生成二维码后，操作员扫码绑定

### 7.4 操作员权限设置
1. 在【个人设置】-【操作员设置】-【操作员权限】
2. 选择操作员
3. 设置操作员权限，包括产品可见范围、操作权限等
4. 保存设置

### 7.5 操作员和团队成员有什么区别？
操作员在平台一切操作均视为主账号操作，团队成员在平台操作均视为子账号操作。
