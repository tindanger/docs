## 个人用户注册

### 注册步骤

1. 在聚千保平台首页点击"会员注册"按钮
2. 填写注册信息：
   - 账号：设置您的登录账号
   - 密码：设置您的登录密码（建议使用8-20位字母、数字组合）
   - 确认密码：再次输入密码确认
   - 联系人姓名：填写真实姓名
   - 手机号码：填写常用手机号码
   - 验证码：点击"获取验证码"按钮获取短信验证码
   - 详细地址：填写您的联系地址

3. 阅读并同意《用户协议》和《隐私政策》
4. 点击"注册会员"按钮完成注册

### 注意事项

<Warning>
- 请确保填写的信息真实有效
- 手机号码将作为重要联系方式和找回密码的验证方式
- 请妥善保管您的账号和密码
</Warning>

### 常见问题

<AccordionGroup>
  <Accordion title="手机验证码收不到怎么办？">
    1. 检查手机是否开启短信拦截
    2. 检查填写的手机号是否正确
    3. 等待1-2分钟后重新获取
    4. 如果仍然收不到，请联系客服处理
  </Accordion>

  <Accordion title="注册时提示账号已存在怎么办？">
    1. 您可能已经注册过账号，请尝试直接登录
    2. 如果忘记密码，可以通过"忘记密码"功能重置
    3. 或者更换一个新的账号名称重新注册
  </Accordion>
</AccordionGroup>

## 企业用户注册

### 企业账号说明

<Note>
企业账号是一个独立的账号体系，专门用于企业管理自身的保险业务。企业通过统一社会信用代码作为唯一标识。
</Note>

#### 账号特点

<CardGroup>
  <Card title="独立性" icon="building">
    - 企业账号完全独立运作
    - 不从属于任何SABC或其他管理账户
    - 专门管理以该企业为投保人的所有订单
  </Card>
  
  <Card title="开通条件" icon="file-check">
    - 完成企业注册信息填写
    - 提交加盖公章的认证材料
    - 平台审核通过后开通
  </Card>
</CardGroup>

#### 功能权限

<Tabs>
  <Tab title="可用功能">
    - 待办事项查看
    - 产品中心浏览
    - 订单管理
    - 站内信息
    - 个人设置
  </Tab>
  
  <Tab title="操作权限">
    - 自主创建投保订单
    - 查看企业所有既往已完成订单
    - 对保期内订单进行批改操作
  </Tab>
  
  <Tab title="使用限制">
    - 产品中心不显示积分比例
    - 订单管理不显示积分信息
    - 不能操作其他会员账号创建的未完成订单
  </Tab>
</Tabs>

### 注册条件

<Info>
根据《金规（2024）9号·关于加强和改进互联网财产保险业务监管事项通知》及相关监管要求，企业用户需要完成认证才能使用平台服务。
</Info>

### 注册步骤

1. **基本信息注册**
   - 单位名称：填写营业执照上的企业全称
   - 证件号码：填写统一社会信用代码
   - 企业账号管理员：填写负责人姓名
   - 身份证号：填写管理员身份证号
   - 管理员手机：填写管理员联系电话
   - 电子邮箱：填写企业联系邮箱

2. **认证材料提交**
   - 下载并填写《聚千保平台企业认证申请书》
   - 盖章后将申请书扫描上传
   - 上传企业营业执照副本
   - 上传企业管理员身份证正反面

3. **阅读并同意相关协议**
   - 《聚千保平台企业认证用户使用协议》
   - 《个人信息保护政策》

### 注意事项

<Warning>
- 企业账号开通后，登录信息将发送至注册时提交的电子邮箱
- 请确保账号管理员为贵单位员工，管理员账号的操作行为均视为贵单位的授权行为
- 企业认证信息必须真实、准确，如有虚假将承担相应法律责任
</Warning>
